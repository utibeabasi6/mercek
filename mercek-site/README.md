# mercek-site

Marketing site for Mercek — a local-first desktop IDE for Amazon ECS. Astro 5 +
Tailwind v4, static-rendered.

## Accuracy mandate

This site markets the Mercek desktop app. **Only Shipped features may appear in
present-tense copy.** Before changing feature claims, re-verify them against the app
codebase and update [`FEATURES.md`](./FEATURES.md) — the verified Shipped / In-progress
/ Planned inventory that all copy on this site is derived from. No invented metrics,
benchmarks, or testimonials.

## Develop

```bash
pnpm install
pnpm dev        # local dev server
pnpm build      # static build to dist/
pnpm preview    # preview the built site
```

## Structure

- `src/layouts/Layout.astro` — `<head>`, SEO/OpenGraph, JSON-LD, no-flash theme script.
- `src/components/` — `Nav`, `Footer`, `AppMock` (HTML recreation of the app window).
- `src/pages/index.astro` — the home page (hero, feature sections, agent, sentinel,
  local-first, install, FAQ).
- `src/pages/docs.astro` — getting-started docs.
- `src/styles/global.css` — design tokens (monospace, dark-default + light) mapped
  into Tailwind v4 `@theme`.
- `public/` — `favicon.svg`, `robots.txt`, `llms.txt` (GEO summary for LLMs).

Sitemap is generated by `@astrojs/sitemap` at build time. Set the canonical host in
`astro.config.mjs` (`site`).
